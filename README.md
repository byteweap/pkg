# pkg 

toolkit for Golang, go version >= 1.18

‍

## 项目目录

* configx: 读取配置文件工具包，如json、toml、yaml等，使用viper
* convert: string 和 []byte高性能转换
* cryptox: AES 和 RSA 加解密
* filex: 文件操作
* floatx: float字符类型公用方法封装
* gis: 有关地理位置相关工具包
* goi18n: go国际化封装包
* golang: go协程安全开启方法
* intx: int类型数据操作方法
* limit: 限速器
* logs: 日志输出到文件并根据时间分割
* mapx: 线程安全的map封装
* mathx: 科学计算
* network: websocket包
* qo: 线性执行封装
* randx: 随机数工具封装
* slices: 切片操作包(范型)
* slicex: 链表结构
* stringx:字符串操作方法
* ternary: 三元表达式
* util: 工具包
* uuid: 32位UUID
* values: 字典类型工具

## 安装

```
go get -u github.com/byteweap/pkg
```
